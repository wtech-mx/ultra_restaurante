A) Instalación - top

0. Recuerden tener instalado xamp version 7.1 en https://www.apachefriends.org SI TIENE HOSTING OLIVIDA ESTE PASO.
1. Subir archivo comprimido de ultraventas a su directorio de servidor y descomprimir allí.
2. Crear una base de datos.
3. Vaya a la aplicación de archivo de base de datos C:\xampp\htdocs\ultraventas\env y complete con los detalles de su base de datos si no aparece es porque esta oculto cambie propiedades para que se muestre carpetas ocultas.
3. servidor\ultraventas\app\.env database.php y complete con los detalles de su base de datos o.

DB_HOST = 'Nombre de host';
DB_USERNAME = 'Nombre de usuario de la base de datos';
DB_PASSWORD = 'Password';
DB_DATABASE = 'Nombre de la base de datos';
4. Importe el archivo sql a su base de datos desde el Carpeta de base de datos.
5. El login esta en la direccion host/nombrecarpeta/public entra ahi para loguearte una vez hecho los anteriores pasos.

C) Múltiples negocios / tiendas: : - top

Configurar varias empresas en la aplicación. No hay restricción en el número de empresas. La información de inventario y contabilidad se guarda por separado para cada negocio.

D) Agregar ubicación / escaparates / bodega: - top

Create multiple locations for your business/shop Manage all of them at the same time. Stocks, Purchases, Sell can be tracked differently for locations. Customize invoice layout, invoice scheme for each location

E) Gestión de usuarios y roles: - top

Potente sistema de gestión de usuarios y roles. Roles predefinidos - Administración y cajero Crear diferentes roles con permiso según su necesidad. Crea usuarios ilimitados con diferentes roles.

F) Contactos (Clientes y Proveedores): - top

Marcar contacto como cliente o proveedor o ambos (cliente y Proveedor) Ver detalles de las transacciones con un contacto. Ver el total del saldo del crédito / débito Defina el término de pago y reciba alertas de pago una semana antes de la fecha de vencimiento.

G) Productos - top

Gestionar productos únicos y variables. Clasificar productos según marcas, categoría, subcategoría. Añadir productos con unidades diferentes. Agregue el número de SKU o genérelo automáticamente con prefijos. Recibe alertas de stock en stock bajo. Ahorre tiempo al calcular automáticamente el precio de venta, el sistema es inteligente para calcular automáticamente el precio de venta según el precio de compra y el margen de beneficio. No es necesario que escriba variaciones cada vez, cree una plantilla de variación y úsela cada vez que necesite crear productos variables.

H)Compras - top

SMS se puede enviar a través de esta aplicación en el módulo SMS.

Añadir fácilmente las compras. Añadir compra para diferentes ubicaciones. Gestionar compras pagadas / vencidas. Obtener notificación de compras vencidas una semana antes de la fecha de pago. Añadir descuentos e impuestos
